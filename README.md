# Sobre o projeto

### API REST construído em NodeJS, utilizando programação orientada a objeto, para facilitar a administração de um PDV, podendo se cadastrar, adicionar categorias, cadastrar clientes, produtos e pedidos

# Funcionalidades

## Usuário
- ### Cadastro
- ### Atualizar dados de cadastro
- ### Usuário logado

## Categorias
- ### Cadastrar categorias
- ### Visualizar categorias cadastradas
- ### Deletar categorias

## Produtos
- ### Cadastrar produtos
- ### Visualizar produtos
- ### Detalhar produtos
- ### Atualizar produtos
- ### Deletar produtos

## Clientes
- ### Cadastrar clientes
- ### Visualizar clientes
- ### Detalhar clientes
- ### Atualizar cliente
- ### Deletar clientes

## Pedidos
- ### Cadastrar pedidos
- ### Visualizar pedidos


# Tecnologias usadas
- ### TypeScript
- ### Node.js
- ### TypeORM
- ### Express.js
- ### JSON

# Para executar este projeto localmente, você precisa:

- ### Conecte um banco de dados
- ### Atribua valores às variáveis de ambiente
- ### Instale dependências para back-end
```shell
npm install
```
- ### Execute o projeto com
```shell
npm run start
```
